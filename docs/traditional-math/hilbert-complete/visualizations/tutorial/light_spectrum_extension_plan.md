def spectral_decomposition(white_light_data, mode):
    """将白光数据按模式分解为特定颜色"""
    if mode == 'φ':

## 🌈 **颜色-模式对应系统**

### **科学的颜色分配**
- **φ模式（红光）**：波长700nm，热烈的创造性
- **e模式（绿光）**：波长550nm，平衡的理性
- **π模式（蓝光）**：波长450nm，深邃的智慧

### **光谱混合效应**
- **黄光**（红+绿）：φ+e模式的观察者
- **紫光**（红+蓝）：φ+π模式的观察者  
- **青光**（绿+蓝）：e+π模式的观察者
- **白光**（红+绿+蓝）：全模式观察者（最高层级）
