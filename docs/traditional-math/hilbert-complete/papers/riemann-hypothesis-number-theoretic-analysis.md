## 4. 间隙与稠密化的数学分析

### 4.1 投影间隙的定义

**定义4.1**（投影间隙）：
在第$k$层，相邻元素间的间隙定义为：
$$\text{Gap}_k(n) = d_k(\text{元素}_{n+1}, \text{元素}_n)$$

其中$d_k$是第$k$层的度量。

**层间间隙关系**：
$$\text{Gap}_{k+1}(n) \leq \text{Gap}_k(n)$$

即下层间隙不大于上层间隙（稠密化效应）。

**定理4.1**（稠密化定理）：

$$\rho_{k+1} \geq \rho_k$$

- **自然数层密度**：$\rho_{\mathbb{N}} = 1$（单位密度）

- **实数层密度**：$\rho_{\mathbb{R}} = \infty$（不可数稠密）

**证明4.1**：基于各层的标准密度理论和投影机制的稠密化性质。$\square$

### 4.3 间隙耗尽的数学条件

**定义4.2**（间隙耗尽）：
当所有层的间隙都达到极小值：
$$\lim_{k \to \infty} \text{Gap}_k = 0$$

但由于递归要求，间隙永不为零：
$$\inf_{k,n} \text{Gap}_k(n) > 0$$

**定理4.2**（间隙耗尽的必然性）：
在有限的投影深度内，间隙必然趋于耗尽。

**证明4.2**：

$$\text{Gap}_{k+1} = f_k(\text{Gap}_k)$$

其中$f_k$是压缩函数，$f_k(x) < x$。

迭代极限：
$$\lim_{k \to \infty} \text{Gap}_k = \lim_{k \to \infty} f_k \circ f_{k-1} \circ \cdots \circ f_1(\text{Gap}_1) = 0$$
$\square$
