# 第二章：坐标系与投影

## 2.1 坐标系的严格定义

### 引言

在母空间$\mathcal{H}$中，ζ函数的不同数学展开（如Dirichlet级数、Euler乘积、Fourier展开等）统一表现为同一对象的不同坐标系表示。本节建立坐标系的严格数学定义，为分析投影遮蔽效应奠定基础。

## 定义 2.1.1 (坐标系)

设$\mathcal{H}$为母空间，$\mathcal{B} = \{e_i\}_{i \in I}$为$\mathcal{H}$的固定标准正交基。

对任意幺正算子$U \in \mathcal{U}(\mathcal{H})$，定义**坐标系**：

$$\mathcal{C}(U) := \{Ue_i : e_i \in \mathcal{B}\}$$

即坐标系是幺正变换$U$作用于标准正交基$\mathcal{B}$所得的新正交基。

## 定理 2.1.1 (坐标系的基本性质)

坐标系$\mathcal{C}(U)$具有以下性质：

1. **正交基性质**：$\mathcal{C}(U)$是$\mathcal{H}$的标准正交基
2. **保范性**：$\langle Uf, Ug \rangle_{\mathcal{H}} = \langle f, g \rangle_{\mathcal{H}}$ 对所有$f,g \in \mathcal{H}$
3. **完备性**：$\overline{\text{span}}\mathcal{C}(U) = \mathcal{H}$

**证明**：

1. **正交基性质**：对任意$i,j \in I$：
   $$\langle Ue_i, Ue_j \rangle = \langle e_i, U^*Ue_j \rangle = \langle e_i, e_j \rangle = \delta_{i,j}$$
   使用了$U$的幺正性$U^*U = I$。

2. **保范性**：直接由幺正算子的定义得出：
   $$\langle Uf, Ug \rangle = \langle f, U^*Ug \rangle = \langle f, g \rangle$$

3. **完备性**：由于$U$是双射且$\mathcal{B}$完备，$\mathcal{C}(U) = U\mathcal{B}$也完备。$\square$

## 定义 2.1.2 (标准坐标系族)

在母空间理论中，我们考虑以下**已知数学展开对应的子空间坐标系**，span相应的闭子空间$\mathcal{H}_\alpha \subset \mathcal{H}$：

**说明**：对于子空间坐标，$\mathcal{C}(U)$是子空间的正交基；对于全$\mathcal{H}$坐标，使用可数稠密集（如$C_c^\infty(\mathbb{R})$逼近）的抽象Gram-Schmidt构造（参考Reed & Simon Vol. I第II.5节，可分空间的基）。

### 1. Dirichlet坐标系
**基向量**：$\{D_n(t)\}$对应ζ函数Dirichlet展开的闭子空间$\mathcal{H}_\zeta \subset \mathcal{H}$。

**数学实现**：首先定义预基$\tilde{D}_n(t) = \frac{n^{-1/2} \exp(-it \log n)}{\sqrt{2\pi}}$，使用Gram-Schmidt过程在$\mathcal{H}_\zeta$中获得正交基$\{D_n(t)\}$（参考Titchmarsh《ζ函数理论》第4章级数边界值）。

**注**：此坐标系span的是Dirichlet多项式的闭包，为$\mathcal{H}$的真子空间。

### 2. Euler坐标系
**基向量**：$\{E_k(t)\}$对应$\log\zeta$展开的闭子空间$\mathcal{H}_{\log\zeta} \subset \mathcal{H}$。

**数学实现**：定义预基$\tilde{E}_k(t) = \frac{1}{k} \sum_p p^{-k/2} \exp(-ikt \log p)$，对于$k=1,2$使用主值求和（作为ζ函数的对数分量），然后使用Gram-Schmidt过程在$\mathcal{H}_{\log\zeta}$中将$\{\tilde{E}_k\}$正交化为$\{E_k(t)\}$（参考Titchmarsh《ζ函数理论》第3章乘积表示）。

**注**：此坐标系span的是von Mangoldt展开的闭包，捕捉ζ函数零点的统计性质但为$\mathcal{H}$的真子空间。

### 3. Fourier坐标系
**基向量**：基于加权正交多项式的谱展开。

**数学实现**：对于权重$w(t) = \frac{1}{\frac{1}{4}+t^2}$，定义预基$\tilde{F}_n(t) = t^n$，然后使用Gram-Schmidt过程相对于加权内积$\int \tilde{F}_m(t) \overline{\tilde{F}_n(t)} w(t)\,dt$获得正交基$\{F_n(t)\}$。

**注**：多项式在此权重下dense于$L^2(\mathbb{R}, w(t)\,dt)$，因为$w(t) > 0$且可积，满足Weierstrass逼近的加权版本。

### 4. 观察者坐标系
**基向量**：基于投影$P_\pm$定义的偶奇分解子空间基，通过偶奇函数基扩展$\{P_+ e_i, P_- e_i\}$，移除零向量并归一化。

**数学实现**：构造子空间基序列：
1. **计算投影**：对标准基$e_i$，先计算$P_+ e_i = \frac{1}{2}(e_i(t) + e_i(-t))$，$P_- e_i = \frac{1}{2}(e_i(t) - e_i(-t))$
2. **移除零向量**：若$P_\pm e_i \neq 0$，则归一化$O_{\pm,i}(t) = \frac{P_\pm e_i}{\|P_\pm e_i\|_{\mathcal{H}}}$（范数相对于加权内积）
3. **收集基向量**：收集所有非零归一化项作为正交基

（参考Rudin《实分析与复分析》第12章投影算子，以及Gram-Schmidt移除冗余）

**通用正交化说明**：
对于非正交预基（如Dirichlet的$\exp(-it \log n)$、Euler的$\tilde{E}_k$、观察者的$\{P_+ e_i, P_- e_i\}$），必须应用Gram-Schmidt正交化过程获得真正的正交基，确保符合定义2.1.1的$\mathcal{C}(U)$形式，其中$U$为相应的幺正算子（参考Reed & Simon Vol. I第II.4节关于正交化的完备性）。

### 5. 全母空间坐标系

**基向量**：$\{G_n(t)\}$为整个$\mathcal{H} = L^2(\mathbb{R}, w(t)\,dt)$的完备正交基。

**数学实现**：对权重$w(t) = \frac{1}{\frac{1}{4}+t^2}$，使用多项式预基$\{\tilde{G}_n(t) = t^n : n = 0,1,2,\ldots\}$，通过Gram-Schmidt过程相对于加权内积获得完备正交基$\{G_n(t)\}$。

**完备性保证**：多项式在此权重下dense，因为$w(t) > 0$且可积，满足加权Weierstrass逼近定理。

**重要澄清**：
1. **坐标系层次**：
   - **子空间坐标**：Dirichlet($\mathcal{H}_\zeta$)、Euler($\mathcal{H}_{\log\zeta}$)、观察者($\mathcal{H}_\pm$)
   - **全空间坐标**：多项式Gram-Schmidt基$\{G_n\}$
2. **完备性分级**：子空间坐标在各自子空间中完备，全空间坐标在整个$\mathcal{H}$中完备
3. **权重兼容性**：只有特殊选择的预基才真正适配权重$w(t)$

**理论意义**：这种分级结构正是"投影遮蔽效应"的数学来源——子空间坐标只能"看到"$\mathcal{H}$的部分结构。

### **关于de Branges坐标的特殊问题**：

de Branges空间$H(E)$使用专门的范数$\int |f(t)|^2 / |E(t)|^2 dt$，其中$E$是指数型整函数（对ζ零点：$E(t) \sim \exp(-\pi|t|/2)$）。这与我们的权重$w(t) = \frac{1}{\frac{1}{4}+t^2} \sim t^{-2}$**不兼容**。

**数学冲突**：
- de Branges权重：$|E(t)|^{-2} \sim \exp(\pi|t|)$（指数增长）
- 母空间权重：$w(t) \sim t^{-2}$（幂律衰减）
- 结果：de Branges基$\{S_n\}$在母空间$\mathcal{H}$中可能不收敛

**发展方向**：
1. **φ/Beatty坐标**：需要基于Beatty序列特性的特殊构造
2. **de Branges坐标**：需要建立与母空间权重的兼容性理论
3. **权重统一**：或考虑修改母空间定义以兼容更多经典坐标系

## 定理 2.1.2 (坐标变换的基本性质)

设$U_1, U_2 \in \mathcal{U}(\mathcal{H})$，则坐标系间的变换$V = U_2 U_1^*$满足：

$$\mathcal{C}(U_1) \stackrel{V}{\longrightarrow} \mathcal{C}(U_2)$$
$$U_1 e_i \mapsto U_2 e_i = V(U_1 e_i)$$

且变换$V$保持所有Hilbert空间结构。

**证明**：
直接验证：$V(U_1 e_i) = U_2 U_1^* U_1 e_i = U_2 e_i$，且$V = U_2 U_1^*$是两个幺正算子的复合，故也是幺正的。$\square$

## 定义 2.1.3 (坐标系下的投影算子)

在坐标系$\mathcal{C}(U)$下，观察者投影算子变换为：

$$P_\pm^{(U)} := U P_\pm U^*$$

其中$P_\pm = \frac{1}{2}(I \pm J)$是母空间中的标准投影。

## 定理 2.1.3 (投影算子的坐标不变性)

坐标系下的投影算子$P_\pm^{(U)}$保持投影性质：

1. **幂等性**：$(P_\pm^{(U)})^2 = P_\pm^{(U)}$
2. **互补性**：$P_+^{(U)} + P_-^{(U)} = I$
3. **正交性**：$P_+^{(U)} P_-^{(U)} = 0$
4. **自伴性**：$(P_\pm^{(U)})^* = P_\pm^{(U)}$

**证明**：
利用幺正算子的性质和原始投影$P_\pm$的性质：

1. $(P_\pm^{(U)})^2 = (UP_\pm U^*)^2 = UP_\pm U^* UP_\pm U^* = UP_\pm^2 U^* = UP_\pm U^* = P_\pm^{(U)}$

2. $P_+^{(U)} + P_-^{(U)} = U P_+ U^* + U P_- U^* = U(P_+ + P_-)U^* = UIU^* = I$

3. 和 4. 类似可证。$\square$

## 推论 2.1.1 (内在量的坐标变换性质)

对任意$f \in \mathcal{H}$和坐标系$\mathcal{C}(U)$：

$$\alpha^{(U)}(f) := \frac{\|P_+^{(U)}f\|^2}{\|f\|^2} = \alpha(U^*f)$$

**证明**：
$$\alpha^{(U)}(f) = \frac{\|P_+^{(U)}f\|^2}{\|f\|^2} = \frac{\|UP_+U^*f\|^2}{\|f\|^2} = \frac{\|P_+U^*f\|^2}{\|U^*f\|^2} \cdot \frac{\|U^*f\|^2}{\|f\|^2} = \alpha(U^*f) \cdot 1 = \alpha(U^*f)$$

其中使用了幺正算子的保范性。$\square$

## 定理 2.1.4 (特殊坐标系的内在量不变性)

**假设**：若幺正算子$U$与反演算子$J$对易，即$UJ = JU$，则：

$$\alpha^{(U)}(f) = \alpha(f) \quad \forall f \in \mathcal{H}$$

**证明**：
若$UJ = JU$，则$UP_+ = U \cdot \frac{1}{2}(I+J) = \frac{1}{2}(U + UJ) = \frac{1}{2}(U + JU) = \frac{1}{2}(I+J)U = P_+U$。

因此：$P_+^{(U)} = UP_+U^* = P_+UU^* = P_+$，故$\alpha^{(U)}(f) = \alpha(f)$。$\square$

**说明**：内在量的完全不变性需要坐标变换与母空间的对称结构兼容。一般情况下，内在量会随坐标变换而变化，这正是"投影遮蔽效应"的数学根源。

## 说明

**坐标系的几何意义**：
- 每个坐标系提供观察母空间的特定"视角"
- 不同坐标系揭示母空间结构的不同方面
- 但所有基本的几何和代数性质（内积、投影、内在量等）在坐标变换下保持不变

**与ζ函数理论的联系**：
- 传统上，ζ函数的不同展开被视为不同的数学对象
- 在母空间框架中，它们统一为同一对象在不同坐标系中的表示
- 这种统一性为理解RH的各种等价判据提供了几何基础

**为后续章节的准备**：
本节建立的坐标不变性是分析"投影遮蔽效应"的基础：虽然内在性质保持不变，但在特定坐标系中观察时，某些结构可能被"遮蔽"而难以直接观察。
