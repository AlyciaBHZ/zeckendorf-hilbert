1. **确定所属领域**：物理、计算、数学、哲学中的哪一类
2. **查找对应表**：使用第二、三、四部分的相关表格
3. **验证等价性**：确认概念在三大定律下的表述一致性
4. **引用定理**：标注对应的等价性证明定理编号

### 5.2 符号使用规范

- **ΨΩΞ符号**：用于表示三大定律的整体框架
- **希腊字母**：用于表示核心数学对象（ρ零点，φ黄金比例等）
- **信息分量**：i₊、i₀、i₋表示三分信息结构
- **数学常数**：φ、e、π表示宇宙基本常数

### 5.3 跨学科交流协议

1. 先用ΨΩΞ统一表述定义概念
2. 再指出在各领域的具体表现
