#### 2.2.1 代码贡献标准

**编程规范**：
```python
# 良好的代码示例
def fibonacci_ratio(n: int) -> float:
    """
    计算第n个Fibonacci比率

    Args:
        n: Fibonacci序号

    Returns:
        第n个比率值

    Raises:
        ValueError: 当n小于1时
    """
    if n < 1:
        raise ValueError("n必须大于等于1")

    if n == 1:
        return 1.0

    fib = [1, 2]
    for i in range(2, n + 1):
        fib.append(fib[i-1] + fib[i-2])

    return fib[n] / fib[n-1]
```

**质量要求**：
- PEP 8代码风格遵守
- 完整类型注解
- 详细文档字符串
- 单元测试覆盖
