```python
def verify_no11_constraint():
    violations = []
    for n in range(1, 100):
        decomp = zeckendorf_decomposition(n)
        # 检查相邻索引
        for i in range(len(decomp)-1):
            if decomp[i+1] - decomp[i] == 1:
                violations.append((n, decomp))
    return violations
```

## 张量积律验证

### ℋ₂ ⊗_Z ℋ₁ ≅ ℋ₃ 的详细验证

**输入空间**：
- B₂ = {00, 01, 10}, dim(ℋ₂) = 3
- B₁ = {0, 1}, dim(ℋ₁) = 2

**拼接过程**：
```
00 ⊗_Z 0 → 000 (合法)
00 ⊗_Z 1 → 001 (合法)
01 ⊗_Z 0 → 010 (合法)
01 ⊗_Z 1 → 011 (禁止: 1|1)
10 ⊗_Z 0 → 100 (合法)
10 ⊗_Z 1 → 101 (合法)
```

**结果验证**：
- 输出：{000, 001, 010, 100, 101}
- B₃ = {000, 001, 010, 100, 101}
- 完全一致 ✓

**维度验证**：
- 理论预测：dim(ℋ₂ ⊗_Z ℋ₁) ≤ dim(ℋ₂) × dim(ℋ₁) = 3 × 2 = 6
- 实际结果：dim(ℋ₃) = 5 < 6 ✓
- 禁11约束确实减少了维度
