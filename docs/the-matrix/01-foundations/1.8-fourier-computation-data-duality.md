### 傅里叶变换的本体机制

**定理1.8.1（本体对偶的数学表示）**：傅里叶变换是计算-数据转换的基础机制。

**证明**：
1. **离散傅里叶变换**：
   $$F_k = \sum_{n=0}^{N-1} s_n e^{-2\pi i k n / N}$$

   将激活序列$(s_n)$（时域计算）转换为频率分量$F_k$（频域数据）

2. **逆变换**：
   $$s_n = \frac{1}{N} \sum_{k=0}^{N-1} F_k e^{2\pi i k n / N}$$

   将数据结构重构为计算过程

3. **守恒性质**：Parseval定理确保能量守恒：
   $$\sum_{n=0}^{N-1} |s_n|^2 = \frac{1}{N} \sum_{k=0}^{N-1} |F_k|^2$$

4. **信息归一化**：在无限维极限下，总信息量守恒为1：
   $$\lim_{N \to \infty} \frac{1}{N} \sum_{n=0}^{N-1} |s_n|^2 = 1$$

   通过Parseval定理，频域也满足归一化（注意系数）：
   $$\lim_{N \to \infty} \frac{1}{N^2} \sum_{k=0}^{N-1} |F_k|^2 = 1$$

因此，傅里叶变换是计算与数据的本体桥梁。$\square$
